Записати в текстовий файл n речень тексту, що задаються програмою на функціональній мові програмування. Речення закінчується символом «точка». Зчитати рядки із створеного програмою файлу, вивести їх на екран. Якщо перше слово речення починається з маленької літери, то замінити її на велику. Записати в новий текстовий файл результат обробки тексту

Код програми

				

-- Завдання № 1
import qualified Data.Char as Char
import Data.List

capitalized :: String -> String
capitalized [] = []
capitalized (head:tail) = Char.toUpper head : lowered tail
  where
    lowered [] = []
    lowered (head:tail) = Char.toLower head : lowered tail

wordsWhen p s =  case dropWhile p s of
                      "" -> []
                      s' -> w : wordsWhen p s''
                            where (w, s'') = break p s'
main = do
  s <- readFile "/uploads/myfiles.txt"
  let a = map capitalized (wordsWhen (=='.') s)
  writeFile "/myfiles/myfiles.txt" (intercalate "." a)
